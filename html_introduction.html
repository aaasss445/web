HTML 소개

HTML이란 무엇인가?

HTML은 웹 페이지 작성을 위한 표준 마크업 언어이다.

  • HTML은 Hyper Text Markup Language의 약자
  • HTML은 웹 페이지의 구조를 설명
  • HTML은 일련의 요소로 구성
  • HTML 요소는 브라우저에 내용을 표시하는 방법을 알려줌
  • HTML 요소는 태그로 표시
  • HTML 태그는 제목, 단락, 표 등과 같은 콘텐츠를 레이블링 함
  • 브라우저는 HTML 태그를 표시하지 않지만 이를 사용하여 페이지의 컨텐츠를 렌더링 함

간단한 HTML 문서

예

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph</p>

</body>
</html>

예시 설명

  • <!DOCTYPE html>선언은 이 페이지를 html로 해석해라... 라는 뜻
  • <html>요소는 해석되는 html의 시작과 끝을 알려줌
  • <head>요소는 문서에 대한 메타 정보를 포함
  • <title>요소는 문서의 제목을 명시
  • <body>요소는 눈에 보이는 페이지의 내용을 포함
  • <h1>요소는 큰 제목을 정의, 숫자를 높일 수록 글자크기는 작아짐, 6까지 있음
  • <p>요소는 문단을 정의

HTML 태그

HTML 태그는 꺾쇠 괄호로 묶인 요소 이름이다.

<tagname>콘텐츠가 여기에 온다...</tagname>

  • HTML 태그는 일반적으로 한 쌍을 이룬다. <p>와 </p>와 같이..
  • 한 쌍의 첫 번째 태그는 시작 태그이고 두 번째 태그는 종료 태그이다.
  • 종료 태그는 시작 태그처럼 작성되지만 태그 이름 앞에 슬래시가 삽입된다.

시작태그와 종료태그는 각각 여는 태그와 닫는 태그라고 불리기도 한다.

웹 브라우저

웹 브라우저 (Chrome, Edge, Firefox, Safari)의 목적은 HTML 문서를 읽고 표시하는 것이다.

브라우저는 HTML 태그를 표시하지 않지만 태그를 사용하여 문서를 표시하는 방법을 결정한다.

HTML 페이지 구조

다음은 HTML 페이지 구조의 시각화이다.

<html>

<head>

<title> 페이지 제목 </title>

</head>

<body>

<h1> 이것은 제목이다 </h1>

<p> 이것은 단락이다 </p>

<p> 이것은 다른 단락이다 </p>

<body>

</html>

<body> 영역(흰색 영역) 안의 내용만 브라우저에 표시된다.

<!DOCTYPE> 선언

<!DOCTYPE>선언은 문서 유형을 나타내며 올바르게 브라우저에 웹 페이즈를 표시하는데 도움이 된다.

페이지 상단 (HTML 태그 이전)에 한 번만 나타내야 한다.

<!DOCTYPE>선언은 대소 문자를 구분하지 않는다.

<!DOCTYPE> HTML5에 대한 선언은 다음과 같다.

<!DOCTYPE html>
