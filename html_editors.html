HTML 편집기

이전 다음

메모장 또는 TextEdit을 사용하여 HTML 작성

전문 HTML 편집기를 사용하여 웹 페이지를 작성하고 수정할 수 있다.

태그가 익숙해 지려면 메모장과 같은 간단한 텍스트 편집기를 사용하는 것이 익숙해지는 데 빠르다.

메모장이나 TextEdit으로 첫 번째 웹페이지를 만들려면 아래 단계를 따라라.

1 단계 : 메모장 열기

Window 8 이상 :

시작 화면을 열고 메모장을 입력한다.

Window 7 이하 :

시작 > 프로그램 > 보조프로그램 > 메모장 열기

2 단계 : HTML 작성

메모장에 HTML을 쓰거나 복사하기

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph</p>

</body>
</html>

3 단계 : HTML 페이지 저장

파일을 컴퓨터에 저장한다. 메모장 메뉴에서 파일 > 다른 이름으로 저장을 선택

파일 이름을 ".html"으로 끝나게 작성하고 인코딩을 UTF-8(HTML 파일의 기본 인코딩)로 설정

.htm 또는 .html을 파일 확장자로 사용할 수 있고 차이는 없다네요

4 단계 : 브라우저에서 HTML 페이지 보기

즐겨 사용하는 브라우저에서 저장된 HTML 파일을 연다.)

  • 저장한 파일 더블 클릭하기
  • 저장한 파일을 오른쪽 클릭해서 연결프로그램으로 브라우저 찾기
  • 브라우저를 실행해서 Ctrl+o 눌러서 저장한 파일 찾아서 열기
