HTML 문서

모든 HTML 문서는 <!DOCTYPE html>선언으로 시작해야 한다

HTML 문서 자체로는 <html>로 시작하고 </html>로 끝난다.

HTML 문서에서 보이는 부분은 <body>와 </body> 사이 이다.

<!DOCYTPE html>
<html>
<body>

<>h1My First Heading</h1>
<>pMy first paragraph</p>

</body>
</html>

HTML 제목

HTML 제목은 <h1>에서 <h6>태그로 정의 된다.

<h1>가 가장 중요한 제목을 정의한다.

<h6>가 가장 중요하지 않은 제목을 정의한다.

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

HTML 단락

HTML 단락은 <p>태그로 정의 된다.

<p>This is a paragraph.</p>
<p>This is another a paragraph.</p>

HTML 링크

HTML 링크는 <a>태그로 정의 된다.

<a href="https://aaasss445.github.io/web/html_introduction.html">This is a link</a>

링크 대상이 href속성에 지정되어 있다.

속성은 HTML 요소에 대한 추가 정보를 제공하는데 사용한다.

나중에 속성에 대해 더 배우게 된다.

HTML 이미지

HTML 이미지는 <img>태그로 정의 된다.

소스 파일(src), 대체 텍스트(alt), 너비(width), 높이(height) 속성으로 제공 된다.

<img src=대충 사진.png alt="대충 나오는 대체 글" width="100" height="150">

HTML 버튼

HTML 버튼은 다음 <button>태그로 정의 된다.

<button>Click me</button>

HTML 목록

HTML 목록은 <ul>(정렬되지 않은/ 글 머리 기호 목록) 또는 <ol>(순서/ 번호가 매겨진 목록) 태그와 <li>태그 (목록 항목)로 정의 된다.

<ul>
   <li>Coffee</li>
   <li>Tea</li>
   <li>Milk</li>
</ul>

<ol>
   <li>Coffee</li>
   <li>Tea</li>
   <li>Milk</li>
</ol
