HTML 스타일 속성

HTML요소의 스타일 설정은 style속성으로 수행 할 수 있다.

HTML style속성의 구문은 다음과 같다.

<tagname style="property:value;">

이 속성은 CSS속성이다. 값은 CSS값이다.

나중에 CSS에 대해서 배운다.

배경색

CSS background-color속성은 HTML요소의 배경색을 정의

이 예제는 페이지의 배경색을 파우더블루로 설정하는 것이다.

<body style="background-color:powderblue;">

<h1>This is aheading</h1>
<p>This is a paragraph</p>

</body>

글자 색

CSS color속성은 HTML 요소의 텍스트 색상을 정의

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red";>This is a paragraph</p>

폰트

CSS font-family속성은 HTML요소에 사용될 글꼴을 정의

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

문자 크기

CSS font-size속성은 HTML요소의 텍스트 크기를 정의

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

텍스트 정렬

CSS text-align속성은 HTML요소의 가로 텍스트 정렬을 정의

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

단원 요약

HTML요소 스타일 지정에 style속성 사용

background-color을 사용하여 배경 색상 지정

color을 사용하여 텍스트 색상 지정

font-family을 사용하여 텍스트 글꼴 지정

font-size을 사용하여 텍스트 크기 지정

text-align을 사용하여 텍스트 정렬 지정
