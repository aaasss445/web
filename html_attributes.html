HTML 속성

속성은 HTML 요소에 대한 추가 정보를 제공한다.

HTML 속성

  • 모든 HTML 요소는 속성을 가질 수 있다.
  • 속선은 요소에 대한 추가 정보를 제공한다.
  • 속성은 항상 시작 태그에 지정된다.
  • 속성은 일반적으로 name = "value"와 같은 한 쌍으로 제공

href 속성

HTML 링크는 <a>태그로 정의 된다.

링크 주소는 href속성에 지정되어 있다.

<a href="https://aaasss445.github.io/web/html_introduction.html">This is a link</a>

<a>태그는 나중에 더 배운다.

src 속성

HTML 이미지는 <img>태그로 정의 된다.

이미지 소스의 파일 이름은 src속성에 지정되어 있다.

<img src="img_man.jpg">

너비와 높이 속성

HTML 이미지도 가지고 있는 width와 height인 이미지의 폭과 높이를 지정하는 속성

<img src="img_man.jpg width="500" height="600">

너비와 높이는 기본적으로 픽셀 단위로 지정된다. 따라서 width="500"은 너비가 500픽셀임을 의미

HTML 이미지 챕터에서 나중에 배운다.

대체 속성

이 alt속성은 이미지를 표시 할 수 없는 경우 사용할 대체 텍스트를 지정

<img src="img_man.jpg" alt="man with a jacket">

이 alt속성은 이미지를 표시할수 없는 경우에도 유용하다. (이미지가 존재하지 않는 경우)

스타일 속성

이 style속성은 색상, 글꼴 크기 등과 같은 요소의 스타일을 지정하는데 사용

<p style="color:red">This is a paragraph</p>

이 속성은 CSS 와 연계되는 부분이다.

lang 속성

문서의 언어는 <html>태그에서 선언 될 수 있다.

언어는 속성으로 선언된다.

접근성 응용 프로그램및 검색 엔진에는 언어 선언이 중요하다.

<!DOCTYPE html>
<html lang="en-US>"
<body>

...
</body> </html>

제목 속성

여기서 title속성이 <p>요소에 추가된다. 단락 위에 마우스를 놓으면 제목 속성의 값이 툴팁으로 표시된다.

<p title="I'm a tooltip">
This is a paragraph
</p>

제안 : 소문자 속성 사용

HTML5 표준에는 소문자 속성 이름이 필요하지 않다.

title 속성은 title또는 TITLE과 같이 대문자 또는 소문자로 쓸 수 있다.

HTML에서 소문자를 권장하고 XHTML과 같은 보다 엄격한 문서 유형에는 소문자를 요구

제안 : 속성 값 인용

HTML5 표준에서는 속성 값 주위에 따옴표가 필요하지 않다.

href에 주소를 적을때 따옴표 없이 쓸수 있다.

하지만 따옴표 없이 적는건 좋지 않다.

XHTML과 같은 보다 엄격한 문서 유형에 대해서는 따옴표를 요구한다.

따옴표를 생략하면 오류가 발생할 수 있다.

작은 따옴표 또는 큰 따옴표?

속성 값 주위에 큰 따옴표가 HTML에서 가장 일반적이지만 작은 따옴표도 사용할 수 있다.

경우에 따라 속성 값 자체에 큰 따옴표가 포함 된 경우 작은 따옴표를 사용해야 한다.

<p title='John "ShotGun" Nelson'>

혹은 그 반대로도...

<p title="John 'ShotGun' Nelson">

단원 요약

  • 모든 HTML요소는 속성을 가질 수 있다.
  • 이 title속성은 추가 "툴팁"정보를 제공
  • 이 href속성은 링크의 주소 정보를 제공
  • width및 height속성은 이미지의 크기 정보를 제공
  • 이 alt속성은 스크린 리더를 위한 텍스트를 제공
  • 속성 이름으로 소문자를 권장

AttributeDescription

title요소에 대한 추가 정보를 지정(툴팁으로 표시됨)
